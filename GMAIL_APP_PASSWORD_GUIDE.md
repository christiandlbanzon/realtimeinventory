# CORRECTED: Gmail App Password Setup

## ✅ Correct Way to Access App Passwords

The direct URL sometimes doesn't work. Use these steps instead:

### Step-by-Step Instructions:

1. **Go to Google Account Security:**
   ```
   https://myaccount.google.com/security
   ```

2. **Enable 2-Step Verification (if not already enabled):**
   - Scroll down to "2-Step Verification"
   - Click "2-Step Verification"
   - Follow the setup process if needed

3. **Access App Passwords:**
   - After enabling 2-Step Verification, scroll down on that same page
   - Look for "App passwords" section (near the bottom)
   - Click "App passwords"

4. **Generate App Password:**
   - Select "Mail" from the dropdown
   - Select "Other (Custom name)"
   - Enter name: "Inventory Alerts"
   - Click "Generate"
   - Copy the 16-character password (remove spaces if any)

### Alternative Method:

If you can't find App passwords, try:

1. Go to: https://myaccount.google.com/
2. Click "Security" in the left menu
3. Under "Signing in to Google", click "2-Step Verification"
4. Scroll down and click "App passwords"
5. Follow steps above

### Troubleshooting:

**"App passwords" option not showing?**
- Make sure 2-Step Verification is enabled
- It may take a few minutes after enabling to appear
- Try refreshing the page

**Still can't find it?**
- Use this direct link (may require sign-in):
  ```
  https://myaccount.google.com/apppasswords
  ```
- Or search Google Account settings for "app passwords"
